Web Clima

Desarrollé un proyecto web que utiliza la API de OpenWeatherMap para mostrar información en tiempo real sobre el clima de diferentes ubicaciones.

La página web proporciona pronósticos climáticos detallados, incluyendo temperatura, descripción del clima y iconos visuales representativos.
Además, implementé una función de búsqueda que permite a los usuarios obtener el clima de una ubicación específica.

Encriptador de texto

Desarrollé una página web interactiva que funciona como un encriptador de texto. La interfaz de la página permite a los usuarios ingresar texto en un campo de entrada. El programa ofrece dos opciones: 'Encriptar' y 'Desencriptar'.

Al hacer clic en el botón 'Encriptar', el programa toma el texto ingresado y lo encripta utilizando un algoritmo específico. Del mismo modo, al hacer clic en el botón 'Desencriptar', el programa revierte el proceso y muestra el texto original.
